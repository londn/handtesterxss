=====PT-BR=====
*O codigo ainda não está aperfeiçoado*
Esse script permite testar payloads de xss "manualmente".
Para isso, ele possui uma opcao "-s", a qual voce deve passar da seguinte maneira:
>python handtesterxss.py <url?param=> -s
Isso faz com que ele mostre todas as linhas do html, permitindo que voce escolha aquelas ou aquela que voce quer testar os payloads.
Apos descobrir as linhas a qual deseja usar, o resto fica facil.
Para usar:
python handtesterxss.py <url?param=> <linhas> <payload>
